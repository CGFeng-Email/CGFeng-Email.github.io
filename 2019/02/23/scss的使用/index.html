scss的使用

发表于 2019-02-23 | 分类于 scss的使用

导入其它scss文件

@import “url”

在scss文件中有一个规定 只要scss文件以下划线开头的就不会自动编译成css文件

_base.scss

scss函数的封装

@mixin username($x,$y){
background-image: url(“…”);
background-size: 200px 200px;
background-repeat: no-repeat;
background-position: $x $y;
}

调用封装的scss函数

@include username(-20px,-20px);

scss嵌套

.top_header {
.top_header_title {
background-color: red;
}
.top_header_content {
background-color: yellow;
}
}

编译后得到的代码：是父子层级的关系 存在不好的地方：css层级太多加载时会浪费性能

优化后的scss嵌套

.top_header {
&_title {}
&_content {}
}

编译后得到的代码：是独一无二的
